Status Management Roadmap

Quick help on how to get around the Status Management application.

Status Management Portal - This page contains information about the application and a link to the Status Management Menu.

Status Management Menu - A menu of status related tasks in the top table and administrative tasks at the bottom.

Add Tasks to Weekly Status Report - The core of the application. Here you add new tasks or edit existing tasks.

Edit Weekly Status Report - A page where you can edit your tasks.

Review and Email Weekly Status Report - Review the entries you've made during the week. Once you're done, you can either send it to yourself as an e-mail or return to further edit.

Time Card Listing - View your time card if you are using the application to manage your time.

Copy Week - Copy the selected week's status entries into the target week.

View Project Listing - View your status reports over a quarter or longer.
