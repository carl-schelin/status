This page is used to provide edit access when editing may not be available from the main application.

The layout is simple if a little bulky.

Each entry consists of three lines. The top line are drop downs. Select the one the entry should be in. The last entry box is the number of 15 minute increments the task tool.

The second line is the task itself. Make any changes to it here.

The third line has the type, the checkboxes, and whether to save or delete the line.

Clicking the save button will update the database with any changes. The three lines will turn green when the database has been updated.

If you want to delete a line, in case you tried to enter a line more than once, you will be prompted to make sure and then the three lines will turn red once the entry has been removed.
